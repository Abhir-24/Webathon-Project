How to declare variables?



In order to declare variables we use the keyword var.
Example:
var a=5;
var a=5,b=5.6;
var p= "I am learning JS";

All kinds of integer values, floating values and strings can be declared using var.


How to Display?



In order to display the value of a variable or a string we use the keyword document.write()for writing on webpage and console.log() for writing a message on chrome console.
Example:
document.write("I am learning JS");//output will come on webpage
var a=5,b=5.6;
console.log(a,b);//values of a and b will be printed on chrome console
var p= "I am learning JS";
