My First Program



We will be writing a simple program to add two numbers.

var a,b,c;
a= parseInt(prompt("enter first no."));//parseInt() is a function that converts the given input into integer form as by default the input taken is in string form
b= parseInt(prompt("enter second no."));//prompt() is a function that takes input from the user
c= a+b;
document.write("The addition of two numbers is "+c);
console.log(c);//will print the value of c in console
